Day 0 · Setup

Before you start: get your environment right

Block 2–3 hours for this. Skipping setup means hitting walls on Day 1 that have nothing to do with AI.

Avoid these Day-0 mistakes

  • Skipping the virtual environment — you'll fight version conflicts all month
  • Hardcoding API keys in your code instead of .env files
  • Pushing to a public GitHub repo with your API key still in it (this is a common, expensive mistake)
  • Using Python 3.8 or earlier — async patterns and type hints behave differently

Glossary — terms decoded

Embedding

A vector of numbers (usually 384–1536 dimensions) that captures the meaning of text. Similar text = similar vectors.

Cosine similarity

A measure between -1 and 1 of how similar two vectors are. 1 = identical direction.

Token

A chunk of text the model processes. Roughly 4 characters or ¾ of a word in English. Billing units.

Context window

Total tokens a model can process in one request. Claude Sonnet 4.6: 200K tokens.

RAG

Retrieval-Augmented Generation. Retrieve relevant docs from a knowledge base, pass to LLM for grounded answers.

BM25

Classic keyword-matching retrieval algorithm. Catches exact terms (product codes, names) that vector search misses.

Cross-encoder

A model that scores a (query, document) pair together. More accurate than embeddings for reranking, but slower.

RRF

Reciprocal Rank Fusion. Algorithm to combine ranked lists from multiple retrieval methods into one merged ranking.

Precision@k

Of the top-k retrieved results, what fraction are actually relevant? Standard retrieval quality metric.

ReAct

Agent pattern: Reason → Act → Observe, loop. The model thinks, calls a tool, sees the result, thinks again.

Faithfulness

RAG eval metric: does the generated answer use only information from the retrieved context? (No hallucinations.)

Prompt caching

Anthropic feature that caches long static prompts so repeated requests cost ~10% of normal.

Prompt injection

Malicious instructions hidden in content the model processes that hijack its behaviour. Treat all external text as untrusted.

MCP

Model Context Protocol. Open standard for connecting LLMs to tools and data sources. Worth learning after Day 30.

How to talk about your capstone in interviews

Q: "Walk me through a project you built recently."

Frame it as a problem, not features. "I built an agent that answers questions over a knowledge base. The interesting part wasn't the agent itself, but making it reliable and cheap enough to actually deploy. I used a model router that sends reasoning steps to a smaller model and only escalates to a bigger one for final answers, which cut cost by about 70%."

Q: "Why hybrid search instead of just vector search?"

Show measurement, not opinion. "I measured both on a 20-question test set. Pure vector search hit precision@5 of around 0.55. Adding BM25 + RRF brought it to 0.71. Adding a cross-encoder reranker brought it to 0.83 — roughly a 50% improvement."

Q: "What would you do differently if you rebuilt this?"

This is the most important question. "I'd add observability earlier — I bolted on structured logging in Week 4 and it would have helped debug Weeks 2 and 3. Self-awareness signals senior thinking."
